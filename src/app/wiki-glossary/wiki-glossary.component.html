Glossary

The spelling database project adopts (and sometimes also adapts) terms originally introduced in LAEME and FITS. This page briefly defines such terms. See the Introduction to LAEME referenced in each entry for a more comprehensive discussion of the terms (click the red icon).

  • form
    An actual spelling variant, which is associated with a specific lexial item and grammatical tag within a LAEME tag. ↗
  • grammel
    The part of LAEME tag which more or less corresponds to grammatical tag in other corpora. Grammels sometimes combine morphological information with functional a positional information. ↗
  • lexel
    The part of the LAEME tag which identifies the lexical item and roughly corresponds to lemma. ↗
  • littera
    This term was originally used in the 5th century "guide" for scribes (Laing and Lass, 2013 II: 2.2.1.) and the authors of LAEME adopted it to be used in discussions of spelling systems. Littera is an abstract object which may be materialized as one of the possible figurae (which are a matter of palaeography rather than orthography) and each littera may have one or more potestates, i.e. sound values which it can represent. For instance, ᵹ (insular ᵹ) and Ȝ (yogh) were two different figurae of the same littera having a few possible potestates including /z/ and /x/. The term littera in the context of the spelling database is applied also to digraphs and polygraphs regarded as relatively fixed (e.g. sh, sch, hw) etc. ↗
  • Litteral Substitution Set (LSS)
    is a set of litterae which may be used to represent a given potestas. For instance, the LSS sc, sh, sch may represent [ʃ]. ↗
  • potestas
    The term is used within the spelling analysis framework to refer to sound values of different litterae ("letters"). ↗
  • Potestatic Substitution Set (PSS)
    is a set of potestates ("sound values") which can be represented by a single littera ("letter"). ↗
  • set
    The term refers to one of the basic units in the database and it was chosen to reflect the connection with Litteral Substitution Set found in LAEME documentation but it has a broader meaning. While LSS in LAEME refers to the set of litterae associated with a certain potestas (sound), sets in the DB are defined as groups of litterae which at least once appear in the same slot (position).
  • slot
    This term refers to a position in an item, which may be filled with various litterae in the individual spelling variants (forms). Every slot is defined by a unique item + its position. For example "FIRE/N (1)" refers to the initial position in the root morpheme of the noun FIRE and the litterae appearing in this slot include f, v, u. The term has been used within other projects involving segmentation of forms, such as From Inglis to Scots (FITS) (see also Kopaczyk et al. 2018).
  • (LAEME) tag
    The basic unit of the LAEME corpus is the tag. Each word in an actual MS is represented by one tag in the corpus. Each tag consists of the actual form found in the MS, the so-called lexel, which serves to identify the lexeme (every lexeme may have multiple forms in the corpus) and a grammel, which is the grammatical tag. ↗
  • text language
    refers to a reasonably homogenous linguistic system associated with a specific piece of text. A single manuscript can be written in multiple text languages corresponding to different scribes or exemplars. ↗
